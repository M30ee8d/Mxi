# M4X-Prime-AI

نموذج دردشة ذكي بلغة JavaScrip (Node.js)، يعتمد موديل GPT‑4o عبر GitHub Proxy، مع إمكانية تغيير نبرة الرد بسهولة.

## التثبيت

```bash
npm install openai readline
```

## التشغيل

أضف متغير البيئة:

```bash
expoi

```

ثم:

```bash
node index.js
```

### طريقة الاستخدام

- اكتب أسئلتك مباشرة، كل سؤال سيرد عليه النموذج.
- لتغيير النبرة:
```
نبرة: ودية
نبرة: حادة
نبرة: رسمية
```
